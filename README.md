# CPSC-6551-Final-Project
Comparing different scoring rules and calibrations on real world weather forecast data.

## Data
https://community.amstat.org/stat-computing/data-expo/data-expo-2018

Due to the size of the files, the data is not included in the repository. The data can be downloaded from the link above and cleaned with the preprocess_data.ipynb (you will have to manually name the output file forecasts_vs_observed.csv).
